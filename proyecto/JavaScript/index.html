JavaScript

JavaScript es un lenguaje de programación de alto nivel que se usa principalmente para crear sitios web dinámicos e interactivos. Es una tecnología clave junto con HTML y CSS, permitiendo funcionalidades como animaciones, formularios interactivos, y la actualización de contenido sin recargar la página. Además de usarse en el lado del cliente (navegador), también puede usarse en el lado del servidor (backend)
